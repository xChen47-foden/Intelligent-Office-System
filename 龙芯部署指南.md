# 龙芯平台部署指南

## 快速部署步骤

### 1. 克隆项目
```bash
git clone https://github.com/xChen47-foden/qxc.git
cd qxc
```

### 2. 后端部署
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r fastapi_app/requirements.txt
python fastapi_app/main.py
```

### 3. 前端部署
```bash
cd ../frontend
npm install
npm run build
```

### 4. 访问应用
- 前端：http://your-ip
- 后端API：http://your-ip:8000
- API文档：http://your-ip:8000/docs
